TLS / Produktion

Für HTTPS auf RHEL 10: sudo bash deploy/install-nginx-tls.sh vcenteremu.local — nginx auf Port 443, Backend intern auf 127.0.0.1:8080, selbstsigniertes Zertifikat.

Beispiel: Session, VM-Liste und Power On

# 1) Session anlegen (Basic Auth)
curl -sk -u "{{ settings.api_username }}:{{ settings.api_password }}" \
  -X POST "{{ api_base }}/com/vmware/cis/session"

# 2) VMs abfragen
curl -sk -H "vmware-api-session-id: <token>" \
  "{{ api_base }}/vcenter/vm"

# 3) VM einschalten (simuliert)
curl -sk -H "vmware-api-session-id: <token>" \
  -X POST "{{ api_base }}/vcenter/vm/vm-web-01/power/start"
